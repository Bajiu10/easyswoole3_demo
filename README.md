# EasySwoole3 Demo

运行项目

> git克隆项目

```
git clone https://github.com/HeKunTong/easyswoole3_demo.git
```

> 安装依赖包

```
composer install
```

> php vendor/easyswoole/easyswoole/bin/easyswoole.php install

tips: 安装生成文件时不要覆盖dev.php和EasySwooleEvent.php文件

> 启动项目

```
php easyswoole start
```
